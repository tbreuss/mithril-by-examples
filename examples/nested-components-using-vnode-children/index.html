This is an example showing nested components using vnode.children.

The example was contributed by tbreuss and last modified on 27 October 2021. Click here to see more examples contributed by the author.

The snippet is using the most current version 2.0.4 of Mithril JS framework. It is aimed at beginners and shows some basic recipes.

In addition to the Mithril hyperscript function m(), here we can see an example of Mithril's m.mount API method. In this example we can also see the usecase of Vnodes (virtual DOM nodes) which is a JavaScript data structure that describes a DOM tree.

Enough said, here it is!

Nested Components Using Vnode Children

Dependencies

Type Name URL
scriptmithril@2.0.4https://unpkg.com/mithril@2.0.4

JavaScript #

const Child = {
  view: function(vnode) {
    return m('div',
      m('h3', vnode.attrs.title),
      m('p', vnode.attrs.text)
    );
  }
};

const Parent = {
  view: function(vnode) {
    return m('div', [
      m('h2', 'Parent Title'),
      vnode.children
    ]);
  }
};

const App = {
  view: function() {
    return m(Parent, [
      m(Child, { title: 'Child Title 1', text: 'Text Number 1' }),
      m(Child, { title: 'Child Title 2', text: 'Text Number 2' }),
      m(Child, { title: 'Child Title 3', text: 'Text Number 3' })
    ]);
  }
}

m.mount(document.body, App);
