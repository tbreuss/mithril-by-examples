Markdown Editor

This example was taken from the official website at https://mithril.js.org/examples.html and slightly modified. It shows a very simple markdown editor using marked.js.

The example was contributed by mithril and last modified on 27 October 2021. Click here to see more examples contributed by the author.

The snippet is using the most current version 2.0.4 of Mithril JS framework. It is aimed at beginners and shows some basic recipes.

Besides Mithril's hyperscript function m() we can see different Mithril API methods like m.trust or m.mount.

Enough said, here it is!

JS #

//model
var state = {
	text: "# Markdown Editor\n\nType on the left panel and see the result on the right panel",
	update: function(value) {
		state.text = value
	}
}

//view
var Editor = {
	view: function() {
		return [
			m("textarea.input", {
				oninput: function (e) {
				  state.update(e.target.value)
        },
				value: state.text
			}),
			m(".preview", m.trust(marked(state.text))),
		]
	}
}

m.mount(document.body, Editor)
