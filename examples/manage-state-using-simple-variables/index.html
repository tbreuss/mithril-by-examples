In Mithril you are free to structure your state data however you'd like, and Mithril takes care of the rest. Below is an example of a simple Counter application written with Mithril.

The example was contributed by kevinfiol and last modified on 19 October 2021. Click here to see more examples contributed by the author.

The snippet is using the most current version 2.0.4 of Mithril JS framework. It is aimed at intermediate users who are familiar with most of the aspects of the framework.

In addition to the Mithril hyperscript function m(), here we can see an example of Mithril's m.mount API method.

Enough said, here it is!

Manage State using Simple Variables

Dependencies

Type Name URL
scriptmithril@2.0.4https://unpkg.com/mithril@2.0.4

JavaScript #

let count = 0;

const App = {
  view: () =>
    m('div',
      m('h1', 'Counter'),
      m('p', count),
      m('button', { onclick: () => count += 1 }, '+'),
      m('button', { onclick: () => count -= 1 }, '-')
    )
};

m.mount(document.body, App);

In this example the state is just a single primitive variable! For small applications, simple widgets or one-off UI components, the above solution is largely sufficient.
